# Extracted UI Sheet Assets

These transparent PNG assets were sliced from:

`frontend/ui_raw/ChatGPT Image 2026年7月4日 12_07_26.png`

The source sheet was segmented by connected alpha regions, then each crop had edge-connected background pixels cleared to transparency. `manifest.json` records the source bounding boxes and output sizes.

These files are prepared for future CSS integration. They are not all wired into the frontend yet.
