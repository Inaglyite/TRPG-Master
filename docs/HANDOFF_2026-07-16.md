## 时间线分支语义

回合记录是“玩家输入 + 守秘人结果”，并保存 `parent_turn_id`。玩家在结果消息上点击分支时，
客户端发送父回合 ID，恢复到本次行动前的决策点：

```text
T1 守秘人给出 A / B / C
└── T2 玩家选择 B，守秘人给出结果

在 T2 上创建分支 → 从 T1 的快照、消息和 choices 建立新 world_id
```

该规则同时适用于实时回合、历史回放、断线恢复和重新叙述。首个没有父回合的开场记录以自身
作为分支源。原时间线不回滚，新分支的状态、存档、日志和笔记独立。
